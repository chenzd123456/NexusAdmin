# Nexus Admin

## 介绍

这是一个前端学习项目，使用了 Vue3、Vite、Pinia、Element-Plus 等技术栈。
项目的目标是实现一个后台管理系统，包含用户管理、角色管理、权限管理等功能。

## 网址

<https://nexusadmin.thiz.top/>

## 团队文档

## 使用方法

1. 克隆项目

    ```bash
    git clone https://github.com/chenzd123456/NexusAdmin.git
    ```

2. 安装依赖

    ```bash
    yarn 
    ```

3. 启动项目

    ```bash
    yarn dev
    ```

4. 打包项目

    ```bash
    yarn build
    ```
